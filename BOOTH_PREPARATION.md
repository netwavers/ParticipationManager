# BOOTH販売準備ガイド

このドキュメントでは、`ParticipationManager` を BOOTH で配布・販売するための手順をまとめています。

## 1. アプリケーションのビルドとパッケージング

販売用の実行ファイルを作成し、配布可能な形式（ZIPなど）にまとめます。

### ビルド手順
`BUILD.md` の手順に従って、Windows用 (`.exe`) および macOS用 (`.app` / `.dmg`) のファイルを生成してください。

```bash
# Windows用
npm run dist -- --win

# macOS用
npm run dist -- --mac
```

### パッケージング
`dist` フォルダに生成されたインストーラーまたは実行ファイルを、ユーザーがダウンロードしやすいようにZIP形式で圧縮します。

- **Windows**: `Participation Manager Setup X.X.X.exe` を右クリック -> 「圧縮」 -> 「ZIPファイル」
- **macOS**: `.dmg` ファイルはそのままで配布可能です。

> [!IMPORTANT]
> 配布ファイルには、必ず `README.md`（使い方）と `LICENSE`（利用規約）を含めることを推奨します。これらをまとめてZIPにすると親切です。

### ウイルスチェック
配布ファイル（ZIP）を [VirusTotal](https://www.virustotal.com/) などのオンラインスキャナーでチェックし、安全であることを確認してください。商品説明に「ウイルスチェック済み」と記載すると安心感を与えられます。

## 2. BOOTH 商品ページの作成

BOOTHの管理画面から「商品の登録」を行います。

### 基本情報
- **商品名**: Participation Manager (参加型配信管理ツール)
- **カテゴリ**: ソフトウェア・ハードウェア > Windows / Mac > ユーティリティ
- **価格**: 任意の価格を設定（無料配布も可能）
- **在庫**: 「0」（ダウンロード商品のため在庫管理不要）

### 商品画像
ユーザーの目を引く魅力的な画像を用意します。
- **メイン画像**: アプリのロゴや、配信画面での使用イメージ。
- **サブ画像**:
    - メイン画面（待機列）のスクリーンショット
    - 設定画面のスクリーンショット
    - モバイル連携画面（スマホUI）のイメージ

### 商品説明
以下の要素を含めると分かりやすいです。

```markdown
YouTube Liveの参加型配信をスムーズに管理できるデスクトップアプリです。

【主な機能】
- 参加希望コメント（"ノ" など）を自動取得して待機列を作成
- 初見さんやHELP希望者の優先表示
- 参加者の自動抽選・選出機能
- 配信画面（OBS）用のオーバーレイ表示
- スマホからのリモート操作機能

【対応OS】
- Windows 10/11 (64bit)
- macOS 10.15以降 (Intel/Apple Silicon)

【使い方】
同梱の README.md をご覧ください。

【利用規約】
- 個人・法人問わず、配信活動でご自由にお使いいただけます。
- 再配布・自作発言は禁止します。
```

### ファイルのアップロード
ステップ1で作成した ZIP ファイルをアップロードします。
- ファイル名にバージョン番号を含めると管理しやすいです（例: `ParticipationManager_v1.0.0_Win.zip`）。

## 3. 法務・サポート体制

### 利用規約 (ライセンス)
トラブル防止のため、簡単な利用規約を定めておきます。
- **免責事項**: 「本ソフトの使用によって生じた損害について、作者は一切の責任を負いません。」
- **禁止事項**: 二次配布、リバースエンジニアリングなど。

### サポート
問い合わせ先を明記します。
- Twitter (X) のDM
- BOOTHのメッセージ機能

## 4. 公開と宣伝

### 公開設定
すべての情報が正しいことを確認し、「公開で保存」をクリックします。

### 宣伝
- **Twitter (X)**: 商品ページのURLと共に、ハッシュタグ（#参加型 #配信素材 #Vtuber素材 など）をつけて投稿します。
- **YouTube**: 自身の配信で使用し、リスナーにアピールします。
