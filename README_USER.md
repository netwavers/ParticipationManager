# Participation Manager

YouTube Liveの参加型配信を強力にサポートする、セッション管理ツールです。

## ✨ 特徴

- **参加者管理**: 「参加希望」コメントを自動で拾ってキューに追加。
- **セッション管理**: 決められた人数を順番、またはランダムに選出。
- **オーバーレイ**: OBS配信画面に現在の状況を表示。
- **安心設計**: 間違って閉じてしまってもデータは保持されます。

## 💻 動作環境

- **Windows**: Windows 10 / 11 (64bit)
- **macOS**: macOS 10.15 以降

## 🚀 インストールと起動

1. **ダウンロード**: BOOTHからダウンロードしたZIPファイルを解凍します。
2. **起動**: フォルダ内の `Participation Manager.exe` (Macの場合は `.app`) をダブルクリックします。
3. **完了**: アプリが立ち上がり、すぐに使用できます。

## 📖 使い方

### 1. 参加者を募る
- **手動追加**: 「Add User」欄に名前を入れて追加します。
- **YouTube連携**: 設定ボタン（歯車アイコン）から「YouTube Integration」を開き、配信のURLに含まれる **Video ID** を設定するだけで、チャットから自動で追加できます（APIキーは不要になりました）。

### 2. セッションを開始する
- **人数設定**: 「Session Size」で参加人数を決めます。
- **スタート**: 
    - `Start Session`: 順番通りに選出
    - `Pick Random`: ランダムに選出
    - `Pick Winner`: 1人だけ抽選（プレゼント企画などに）

### 3. OBSに表示する (オーバーレイ)
配信画面に参加者リストやタイマーを表示できます。

1. アプリ右上の **"OBS Overlay"** ボタンを押します。
2. 新しいウィンドウで背景が透明なオーバーレイ画面が開きます。
3. **OBS Studio** 側で「ウィンドウキャプチャ」ソースを追加し、このウィンドウを指定してください。
   - ※ ウィンドウの背景は黒く見えることがありますが、OBSで「カラーキー」フィルタなどを使うと透過できる場合があります（環境によります）。
   - ※ または、OBSの「ブラウザ」ソースを使い、ローカルファイルとして `obs.html` を指定する方法もありますが、ウィンドウキャプチャが一番簡単です。

### 4. 終了と履歴
- プレイが終わったら、リストのチェックボタン (✔) を押すと「履歴」に移動します。
- もう一度参加したい場合は、履歴から「リターン」ボタン (↩) で列に戻せます。
